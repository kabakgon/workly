# Screenshots - Workly 2.0

## Instrukcja dodawania screenshotów

1. **Zrób screenshoty** następujących stron:
   - Strona logowania
   - Dashboard
   - Lista projektów
   - Szczegóły projektu
   - Lista zadań
   - Diagram Gantta (mockup)
   - Zarządzanie zależnościami

2. **Zapisz screenshoty** w folderze `docs/screenshots/` z nazwami:
   - `login.png` - Strona logowania
   - `dashboard.png` - Dashboard
   - `projects-list.png` - Lista projektów
   - `project-detail.png` - Szczegóły projektu
   - `tasks-list.png` - Lista zadań
   - `gantt.png` - Diagram Gantta
   - `dependencies.png` - Zarządzanie zależnościami
   - `theme-switcher.png` - Przełącznik motywów (dropdown z opcjami Synthwave/Jasny)

3. **Zoptymalizuj obrazki** (opcjonalnie):
   - Użyj narzędzi jak TinyPNG lub ImageOptim
   - Zalecany rozmiar: max 1920x1080px
   - Format: PNG lub JPG

4. **Dodaj do README.md** - screenshoty są już dodane w sekcji "Screenshots"
